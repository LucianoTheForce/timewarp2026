# Pasta de Músicas

Coloque seus arquivos de música nesta pasta para usar no sistema audio-reativo.

## Formatos Suportados
- MP3
- WAV
- OGG

## Como Adicionar Músicas

1. Coloque seus arquivos de música nesta pasta
2. Renomeie os arquivos para:
   - `track1.mp3`
   - `track2.mp3`
   - `track3.mp3`
   - etc.

3. Ou edite o arquivo `js/audioSystem.js` para adicionar caminhos personalizados

## Exemplo de Estrutura

```
musicas/
  ├── track1.mp3
  ├── track2.mp3
  ├── track3.mp3
  └── README.md
```

As músicas aparecerão automaticamente no painel de controle da interface.
